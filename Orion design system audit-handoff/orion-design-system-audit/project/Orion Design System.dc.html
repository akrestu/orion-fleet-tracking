01 · Foundations

A calm, safety-first system for 24/7 ops.

Orion runs in control rooms and cabs. The palette stays quiet so that signal colors mean something: green/amber/red are reserved strictly for device status, never decoration. Type is tuned for dense telemetry with tabular numerals.
